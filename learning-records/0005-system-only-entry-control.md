# 系统内交易与进场控制

用户在 2026-07-02 明确提出：以后不再做系统外交易，当前最需要严格控制进场。

后续教学和复盘中应默认采用以下规则：

- 系统外交易默认禁止，不再把“偶尔小玩一下”当作可接受行为
- 下单前优先确认是否属于用户自己的主系统，而不是先讨论这笔可能赚多少
- 当前阶段的重点不是增加交易机会，而是减少错误入口
- 交易前检查单应尽量短，主要用于确认三件事：系统、触发、失效

用户当前认可的极简检查：

```text
1. 是我的系统吗？
2. 触发完成了吗？
3. 错了在哪里退出？

三条有一条答不上来，不下单。
```

需要特别拦截的系统外入口：

- 为了吃资金费进场
- 亏损后反手同一标的
- 临时做震荡小波段
- 看到异动后凭感觉追单
- 因为想修复亏损而寻找机会

本记录只用于学习、研究和复盘，不构成投资建议。
