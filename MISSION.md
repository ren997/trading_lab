# Mission: 趋势交易能力提升

## Why

我已经在币圈、美股和 A 股进行过一段时间的主观趋势交易，主要做多，关注箱体突破追多、杯柄形态，以及上升趋势中回踩均线后的延续机会。过去有过明显亏损，现在大致能做到不亏或小赚；这个工作区的目标，是把当前水平进一步推进到“更稳定、更可解释、更可复盘”的阶段，而不是继续依赖感觉、临场情绪或单次行情运气。

## Success looks like

- 能把自己的几类常用 setup 拆开统计，例如箱体突破、杯柄突破、均线回踩，而不是混在一起凭印象判断
- 能清楚解释仓位、止损、盈亏比、胜率、期望值和最大回撤之间的关系，并把这些指标用于实战复盘
- 能用固定模板记录一次交易计划、执行结果和复盘结论，区分“看对但没赚到”和“方法本身没有优势”
- 能用 Python 对至少一个常见做多形态做最小研究或回测，并说明数据、手续费、滑点和市场差异如何影响结果
- 能逐步提升执行稳定性和情绪控制，减少冲动追单、拿不住、提前止盈和亏损后失衡操作

## Constraints

- 教学语言：中文
- 技术路线：优先使用 Python 和 Markdown
- 当前经验：已有真实交易经验，能大致做到不亏或小赚，但还未形成稳定、可量化的优势验证流程
- 学习方式：小步推进，每次都尽量留下可复习文件
- 风险边界：以教育、研究和模拟验证为主，不提供个性化投资建议

## Out of scope

- 荐股、喊单、短期暴富叙事
- 在没有验证流程前直接扩大仓位、提高杠杆或增加高频交易
- 把单次回测、单个形态样本或少量盈利交易包装成稳定可复制的收益承诺
