# Trading Lab

这是一个面向长期学习的交易研究工作区，用来系统学习交易基础、风险管理、复盘方法和 Python 回测实践。

这里的内容以学习、记录和研究为目标，不构成荐股、喊单、投资承诺或任何个性化投资建议。

## 学习目标

- 建立对交易基础概念的清晰理解，例如收益风险、仓位、止损、回撤和交易成本。
- 用结构化方式记录复盘，减少凭感觉下单和事后合理化。
- 用 Python 编写最小可运行的回测脚本，理解数据、信号、仓位和绩效评估之间的关系。
- 逐步形成“先验证、后表达观点”的研究习惯。

## 目录

- `lessons/`：按编号递增的课程讲义，默认使用 Markdown。
- `reference/`：可长期复用的参考资料，例如复盘模板、回测检查清单、术语表。
- `learning-records/`：关键学习记录，用于保存已经确认掌握的知识和纠正过的误区。
- `assets/`：复用素材，例如样式、图表模板、示例数据说明。
- `.agents/skills/teach/`：针对本仓库定制的教学 skill。
- `MISSION.md`：这个工作区为什么存在，以及“学会以后能做到什么”。
- `RESOURCES.md`：高信任度学习资源清单。
- `NOTES.md`：教学偏好和过程备注。
- `AGENTS.md`：仓库协作约定。

## 目录约定

- `lessons/` 只放课程内容，文件名使用 `0001-xxx.md` 这种编号格式。
- `reference/` 只放跨课程复用的资料，不放日常随手笔记。
- 当某节课进入实操时，再创建对应的顶层编号实践目录，例如 `01_交易基础/`、`02_风险管理/`、`03_回测入门/`。
- 所有策略讨论都默认补充前提、风险、局限和验证条件。

## 当前学习重心

- 风险管理优先于策略花样
- 复盘优先于预测
- 回测优先于实盘
- Python 优先于手工表格

## 继续使用

如果你使用支持本地 skill 的 Agent，可以在本仓库里继续基于 `teach` skill 学习，例如：

```text
/teach 继续教我交易中的仓位管理
```
