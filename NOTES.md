# Teaching Notes

- 默认使用中文讲解。
- 优先使用 Python 做示例和练习。
- 用户已有真实交易经验，涉及币圈、美股、A 股，主要做多。
- 常见关注 setup：箱体突破追多、杯柄形态、上升趋势中回踩均线后的延续机会。
- 用户曾经有过较明显亏损，目前大致能做到不亏或小赚，目标是进一步提升稳定性与理解深度。
- 默认先学风险管理、复盘、回测和执行控制，再讨论更复杂的策略构造。
- 课程与参考资料默认使用 Markdown，方便长期维护。
- 每次学习尽量留下课程、参考资料、代码或学习记录，确保下次可以接着学。
- 所有交易相关内容都要明确：这不是投资建议，重点是方法、验证和风险意识。
- 对量价形态的学习，重点不应只是“记形态名字”，而要拆成可记录、可统计、可验证的条件。
