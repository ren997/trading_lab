# 0001 从不亏小赚到稳定进步

## 这节课为什么先学这个

你现在最关键的阶段，不是继续收集更多形态名字，而是把“偶尔做对”推进成“知道自己为什么能做对，也知道什么时候别做”。  
过去你已经交过学费，现在又逐渐能做到不亏或小赚，这通常说明问题已经不再只是方向判断，而是进入了更难的一层：**优势有没有被稳定表达出来**。

这一课的目标，是先判断你接下来最值得补的能力层，而不是急着讨论某个具体买点。

## 先给你的阶段判断

基于你提供的信息，我的判断是：

1. 你已经不是“零基础”，而是有一定盘感和形态观察经验的人。
2. 你现在最缺的，大概率不是再多学 10 个形态，而是把已有 setup 做成可区分、可复盘、可统计的系统。
3. 你现在的提升空间，主要来自三个层面：
   - 风险控制是否稳定
   - setup 边界是否清楚
   - 执行和心态是否可复盘

换句话说，你下一阶段更像是在做“交易系统整理”，不是在做“形态收藏”。

## 你最需要提升的 5 个方向

### 1. 把 setup 分开，不要混着评估

你现在提到的模式至少有三类：

- 箱体突破追多
- 杯柄形态做多
- 上升趋势中回踩均线再走强

这三类 setup 的市场结构、失败方式、止损逻辑、适合的波动环境都不一样。  
如果你把它们混在一起看，只会得到一种模糊感觉，例如“最近好像还行”或者“我这个月做得很差”，但你不知道到底是哪一类在赚钱，哪一类在拖后腿。

你接下来最该做的第一件事，不是优化进场，而是把历史交易按 setup 分类。

最少先分成这三栏：

- `breakout_range`：箱体突破
- `cup_handle`：杯柄突破
- `pullback_ma`：上升趋势回踩均线

然后分别统计。

### 2. 从“看胜率”升级到“看 R 和回撤”

很多做到不亏或小赚的人，会卡在一个阶段：方向判断经常没错，但一旦亏的时候亏得太大，或者赚的时候拿不住。

你接下来必须开始系统地看这几个指标：

- 单笔风险 `1R` 是多少
- 平均盈利是多少 `R`
- 平均亏损是多少 `R`
- 胜率是多少
- 期望值是否为正
- 连续亏损时最大回撤会到哪里

如果没有这些数据，所谓“我最近有进步”通常只是感觉，不是证据。

### 3. 把“形态学习”改成“条件学习”

你提到想继续学量价形态，比如杯柄形态。  
这里最容易掉进去的坑是：学得越多，越容易在图上“看什么都像”。

所以后面学形态时，重点不要放在名字，而是放在条件：

- 前面那段上涨是否足够强
- 杯体是否太深
- 柄部是否过长或过深
- 突破时量能是否真正放大
- 突破位置是否离前高过远
- 所处的大盘 / 板块 / 币种环境是否支持追多

也就是说，**你不是要学“杯柄长什么样”，而是要学“什么样的杯柄值得做，什么样的不值得做”。**

### 4. 把心态问题翻译成可操作规则

“心态控制”这个词很容易说得很空。  
真正有效的做法是把情绪问题翻译成规则问题。

例如下面这些情况：

- 追高后害怕回撤，提前卖出
- 连亏两笔后急于翻本
- 本来没到买点，因为怕错过而提前进
- 赚一点就跑，亏的时候反而扛

这些表面上是心态，底层通常对应：

- 仓位过大
- 止损位置不清楚
- 进场条件不够明确
- 没有“今天到此为止”的规则
- 没有把单笔交易放回样本，而是太在意这一次输赢

所以你的“心态课”，应该优先从以下规则入手：

- 单笔最大风险固定
- 单日最大亏损固定
- 连续亏损多少笔后强制停手
- 没有计划的单子不做
- 交易后必须写复盘

### 5. 分市场看，不要把币圈、美股、A 股当同一种游戏

这三类市场差别很大：

- 波动特征不同
- 交易时段不同
- 流动性结构不同
- 消息驱动强度不同
- 假突破和回撤深度也不同

同一个 setup，在币圈、美股、A 股里的表现可能完全不一样。  
所以后面你做复盘或统计时，至少还要加一列：

- `crypto`
- `us_stock`
- `a_share`

这样你才能知道：

- 是 setup 本身有问题
- 还是这个 setup 只适合某个市场

## 现阶段最值得优先补的能力排序

如果让我按优先级给你排，我会建议你按这个顺序提升：

1. 风险管理
2. 交易记录与复盘质量
3. setup 拆分统计
4. 执行纪律和心态控制
5. 形态细化学习
6. Python 回测 / 研究

原因很简单：

- 没有风险管理，学再多形态也可能被几次错误操作打回去。
- 没有记录和统计，你无法知道自己到底该保留什么、砍掉什么。
- 没有执行纪律，正确的方法也会被错误执行拖垮。
- 形态学习应该建立在前面这些地基之上。

## 接下来两周建议你做的最小动作

先不要同时做太多改动。你只需要完成这 4 件事：

1. 把最近 30 笔交易按 `setup + market` 分类
2. 用 `reference/复盘模板.md` 补写其中最有代表性的 10 笔
3. 给每笔交易增加 `计划风险 / 实际亏损 / 是否按计划执行` 三列
4. 写出你最常见的 3 个错误动作

如果你能把这一步做完，我们下一步就能非常具体地判断：

- 你到底是形态识别问题
- 风控问题
- 还是执行与心态问题

## 一个先不要急着做的事

先不要急着扩大仓位。  
当一个交易者刚从“长期亏损”进入“不亏或小赚”阶段时，最危险的事情之一，就是误把“阶段性恢复”当成“稳定优势已经形成”。

美国 NFA 的投资者最佳实践明确强调，高波动交易只能使用能承受损失的风险资本；Investor.gov 也把风险定义为投资决策中的不确定性与潜在损失。[NFA](https://www.nfa.futures.org/investors/investor-resources/files/investor-best-practices.html) [Investor.gov](https://www.investor.gov/introduction-investing/investing-basics/what-risk)

我的推断是：你现在最值得追求的，不是“更快赚更多”，而是“先证明自己真的有稳定优势”。

## 这节课的结论

你当前最需要提升的，不是单纯再学几个新形态，而是把已有交易经验沉淀成一套更稳定的流程：

- setup 分开统计
- 风险统一表达
- 复盘聚焦执行与偏差
- 分市场验证
- 再把最常用的形态做成可研究的条件

只有这样，后面学杯柄、量价关系和心态控制，才会真正进入“能落地”的阶段。
