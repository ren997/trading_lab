# 0003 交易触发、试仓与确认仓位

## 学习目标

这一课要解决一个很容易反复犯的问题：不要把“我看好”直接等同于“我可以买”。

学完以后，你要能把一笔交易拆成四个状态：

```text
观察池 → 触发 → 试仓 → 确认仓位
```

这节课直接服务于你的 `MISSION.md`：把主观趋势交易推进到“更稳定、更可解释、更可复盘”，尤其减少冲动追单、提前入场和止损变宽。

## 为什么现在学这个

你最近记录的斯达半导这笔交易很典型：

```text
周线突破长期震荡区，这是大周期背景。
小时级别有杯柄雏形，这是小周期潜在触发。
但你自己也意识到：应该等小时级别突破更好，止损更小。
```

这里的问题不是你用了多周期，而是周期分工不够硬：

```text
周线给了你信心；
小时线还没完成触发；
你已经提前承担了完整风险。
```

这一课就是为了给这种情况加一道闸门。

## 必要概念

### 1. 观察池

观察池的意思是：这个标的值得关注，但还不等于可以买。

例如：

- 周线突破长期箱体
- 日线重新站上关键均线
- 板块开始走强
- 量价结构出现改善

这些都可以让一个标的进入观察池。

但请记住：

```text
进入观察池 ≠ 入场信号
```

观察池只回答一个问题：

```text
这东西值不值得我继续盯？
```

它不回答：

```text
我现在能不能买？
```

### 2. 触发

触发是具体、可验证、可失败的入场条件。

好的触发应该能被写成一句话：

```text
如果出现 X，我才允许入场；如果没有 X，我继续等待。
```

例如：

- 小时级别杯柄突破柄部高点
- 回踩均线后重新放量站上前一根关键 K 线高点
- 箱体突破后回踩不破，再次转强
- 突破后没有快速跌回箱体

触发必须比“看起来不错”更具体。

### 3. 试仓

试仓不是正式入场，它是承认自己还没有完整触发时的低风险表达。

如果你确实怕错过，可以允许自己试仓，但要用很小的风险：

```text
试仓风险：0.25R
正式完整风险：1R
```

试仓的目的不是赚钱，而是防止自己因为完全空仓而焦虑。

如果试仓后触发没有完成，不能继续加；如果结构变坏，应该按试仓计划退出。

### 4. 确认仓位

确认仓位是指：触发真正发生后，才允许把风险补到完整计划。

例如：

```text
周线突破是背景；
小时杯柄突破是触发；
未突破前最多 0.25R 试仓；
突破后才允许补到 1R。
```

这套结构能防止你把“提前一点”变成“提前重仓”。

## 多周期交易的分工

你现在最适合使用这条规则：

```text
大周期决定要不要看；
小周期决定什么时候买；
止损跟随入场周期；
盈利后才考虑升级持仓周期。
```

例如：

```text
周线：长期箱体突破，加入观察池。
小时：等待杯柄突破，作为入场触发。
止损：若按小时入场，就用小时结构止损。
仓位：按小时止损距离反推数量。
管理：如果顺利走强，再考虑把剩余仓位升级为周线波段持有。
```

反过来，如果你选择做纯周线交易，也可以，但规则不同：

```text
入场：周线突破或周线回踩确认。
止损：周线结构失效。
仓位：更小。
持仓：更长。
不要因为小时级别噪音频繁改变计划。
```

两种做法都可以，不能混着来。

## 一个最小练习

用你的斯达半导这笔交易，补写下面 5 句话：

```text
1. 我的观察池理由是：周线级别突破震荡区,物理ai未来会很吃香,这只股还没有启动,而且周线的突破是有很大成交量的
2. 我的正式入场触发应该是：在小级别寻找入场条件,小时级别有一个杯柄形态的雏形
3. 在触发完成前，我最多允许自己承担多少 R：0.25r
4. 如果按小时级别入场，我的失效条件是：杯柄形态失效
5. 如果我选择做纯周线交易，我的止损和仓位应该如何调整：止损不变 仓位更小
```

写完以后，把结论补到这笔交易的备注或复盘截图里。

## 交易前检查清单

每次下单前，先问自己：

```text
1. 这笔交易的主周期是什么？
2. 大周期只是背景，还是已经是入场信号？
3. 小周期触发是否已经完成？
4. 止损跟随哪个周期？
5. 现在是试仓，还是正式完整仓位？
6. 如果还没触发，我是否把风险限制在 0.25R？
7. 如果这笔马上亏损，我会不会开始改口说“我是做大周期的”？
```

第 7 个问题特别重要。  
如果答案是“可能会”，说明你的交易计划还不干净。

## 常见错误

### 错误 1：用大周期增强信心，用小周期提前入场

这会导致你总觉得“机会很好”，但实际触发还没发生。

### 错误 2：用小周期入场，却用大周期硬扛

如果你按小时图入场，小时图失效后却说“周线还好”，这就是临场换规则。

### 错误 3：试仓变成完整仓位

试仓必须提前规定风险。如果没有规定，它就不是试仓，只是提前入场的漂亮说法。

## 下一步建议

从下一笔开始，你的交易日志里至少要写清楚三句话：

```text
观察池理由：
入场触发：
止损周期：
```

如果还没触发，但你还是想参与，只允许写：

```text
交易状态：持仓中
仓位性质：试仓
计划风险：0.25R
```

等你连续记录 20 到 30 笔，我们再按 setup 统计：

- 哪类交易真正等到了触发
- 哪类交易经常提前入场
- 提前入场的平均 R 是否明显更差
- 试仓有没有被你偷偷做成完整仓位

## 风险提醒

这里的内容只用于学习、研究和模拟验证，不构成个性化投资建议。多周期结构、试仓和 R 倍数只是复盘工具，不能保证收益，也不能替代真实风险控制。
