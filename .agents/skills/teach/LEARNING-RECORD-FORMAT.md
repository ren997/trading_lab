# Learning Record 格式

学习记录位于 `learning-records/`，文件名使用递增编号，例如 `0001-xxx.md`、`0002-xxx.md`。

它们的作用不是写流水账，而是记录会影响后续教学的重要事实。

## 模板

```md
# {简短标题}

{1 到 3 句话：说明学到了什么、纠正了什么误区，或者确认了什么既有能力，以及它为什么会影响下一步学习。}
```

## 什么时候应该写

1. 用户证明自己真正理解了一个非平凡概念
2. 用户明确说出了已有基础，后续不必重教
3. 一个重要误区被纠正了
4. 学习目标因为新的认识而发生变化

## 什么时候不该写

- 只是“今天讲过了什么”
- 纯过程日志
- 还没有证据表明用户真的掌握了
