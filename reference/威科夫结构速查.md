# 威科夫结构速查

用途：把威科夫语言翻译成交易复盘中能记录、能验证的结构假设。这里的内容只用于学习、研究和模拟验证，不构成投资建议。

## 一句话理解

威科夫不是“看到某个图形就买卖”，而是用价格、成交量、时间和位置来判断：

```text
当前区间更像吸筹、派发、再吸筹，还是再派发？
突破前后，供需关系有没有真的改变？
我的入场点是否有清楚的失效位置？
```

## 三个核心法则

### 1. 供给与需求

需求强于供给，价格更容易上涨；供给强于需求，价格更容易下跌。

复盘时不要只写“放量”，要写清楚：

```text
放量后价格有没有推进？
下跌时成交量是放大还是缩小？
支撑附近是否出现卖压衰竭？
阻力附近是否出现买盘耗尽？
```

### 2. 因与果

长期横盘或区间震荡可能是在积累后续趋势的“原因”，突破后的趋势是“结果”。

但这不是保证。区间越长，只能说明潜在能量更值得关注，不能说明一定向上。

### 3. 努力与结果

成交量可以理解为努力，价格推进可以理解为结果。

典型问题：

```text
很大成交量却无法继续上涨，是不是上方供给很重？
很大成交量却不再创新低，是不是下方有承接？
缩量回踩后重新转强，是不是供应减少？
```

## 吸筹区间的常见事件

| 缩写 | 含义 | 复盘时关注 |
| --- | --- | --- |
| PS | 初步支撑 | 下跌后第一次出现明显承接 |
| SC | 卖出高潮 | 恐慌抛压集中释放，常伴随大波动和大成交 |
| AR | 自动反弹 | 抛压释放后自然反弹，常用于定义区间上沿 |
| ST | 二次测试 | 回到低位附近测试供应是否减弱 |
| Spring | 跌破区间后的快速收回 | 假跌破后是否重新站回区间 |
| SOS | 强势信号 | 放量、价差扩大、突破区间或关键阻力 |
| LPS | 最后支撑点 | SOS 后回踩不深、缩量、守住关键位 |

## 吸筹区间的五个阶段

| 阶段 | 核心作用 | 常见事件 | 复盘重点 |
| --- | --- | --- | --- |
| Phase A | 停止原来的下跌趋势 | PS、SC、AR、ST | 供应是否从绝对主导变成开始被承接 |
| Phase B | 建立“因” | 多次 ST、区间内宽幅震荡 | 大资金是否在区间内吸收供应，后半段下跌量能是否减少 |
| Phase C | 测试剩余供应 | Spring、Shakeout、Test，也可能没有 Spring | 跌破后是否快速收回，测试时成交量和价差是否收缩 |
| Phase D | 需求开始占优 | SOS、LPS | 上涨是否更顺畅，回踩是否更浅、更缩量 |
| Phase E | 离开交易区间 | 突破、Markup、再吸筹 | 突破后是否被市场接受，回踩是否不轻易跌回旧区间 |

使用时要注意：Phase A-E 是阅读结构的地图，不是固定交易流程。真实走势可能缺少 Spring，也可能在 Phase B 里反复震荡很久。

## 派发区间的常见事件

| 缩写 | 含义 | 复盘时关注 |
| --- | --- | --- |
| PSY | 初步供给 | 上涨后第一次出现明显卖压 |
| BC | 买入高潮 | 追涨需求集中释放，常伴随大波动和大成交 |
| AR | 自动回落 | 买盘减弱后自然回落，常用于定义区间下沿 |
| ST | 二次测试 | 回到高位附近测试需求是否减弱 |
| UT / UTAD | 上冲失败 / 派发后上冲 | 假突破后是否跌回区间 |
| SOW | 弱势信号 | 放量、价差扩大、跌破区间或关键支撑 |
| LPSY | 最后供给点 | SOW 后反弹无力，需求不足 |

## 和你当前系统的对应关系

### 箱体突破

威科夫视角下，箱体突破不是只看“突破了没有”，而是先问：

```text
这个箱体更像吸筹还是派发？
突破前有没有供应减弱或需求增强？
突破后有没有快速跌回箱体？
回踩时是缩量承接，还是放量砸回？
```

### 杯柄形态

杯柄可以暂时理解为一种更具体的再吸筹结构。

更好的杯柄通常需要：

```text
左侧上涨有强度；
杯体回撤不过深；
柄部缩量、波动收窄；
突破时需求重新增强；
失败点清楚。
```

### 均线回踩

均线本身不是威科夫核心，回踩的关键是：

```text
回踩时供应是否减少？
再次转强时需求是否恢复？
止损是否能放在结构失效点，而不是随意放在均线下方？
```

## 下单前三问版

```text
1. 这是我的箱体突破、杯柄突破或均线回踩系统吗？
2. 威科夫结构只是背景，还是已经完成我的入场触发？
3. 如果判断错了，跌破/站回哪里说明结构失效？
```

三条有一条答不上来，不下单。
